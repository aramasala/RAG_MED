# Scripts

Helper scripts for RAG_MED (run from project root).

| Script           | Description |
|-----------------|-------------|
| `clean_pdf.sh`  | Clean one or more PDFs; usage: `./scripts/clean_pdf.sh file.pdf` or `./scripts/clean_pdf.sh dir/` |
| `generate_qa.sh`| Generate QA from a PDF; usage: `./scripts/generate_qa.sh file.pdf [--num-questions N]` |

Ensure the virtualenv is activated (`source .venv/bin/activate`) or use the wrapper that activates it.
